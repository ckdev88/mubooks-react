# Clone of Mubooks written with the following technologies:
- Main library:			React 
- Routing:				React router
- Types:				TypeScript 
- Build tool:			Vite
- Compiler:				SWC
- Runtime & bundler:	Bun
- Package manager:		Bun
- Testing:				Jest
- User data handling:	Supabase & PostGreSQL
- Styling:				Vanilla CSS (Optimize w/PostCSS later)
- Data:					JSON
- Data sanitation:		Shell script
