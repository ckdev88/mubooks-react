# Mubooks

## Technologies
- Main library:			React 
- Routing:				React router
- Types:				TypeScript 
- Build tool:			Vite
- Compiler:				SWC
- Runtime & bundler:	Bun
- Package manager:		Bun
- Testing:				Jest
- User data handling:	Supabase & PostGreSQL
- Styling:				Vanilla CSS (cleanup by purgeCSS)
- Data:					JSON
- Deployment:		    Shell script
